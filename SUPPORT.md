# Support

## How to get help

- **Bug reports**: [Open an issue](https://github.com/gawwr4v/RadialMenu/issues/new?template=bug_report.md)
- **Feature requests**: [Open an issue](https://github.com/gawwr4v/RadialMenu/issues/new?template=feature_request.md)
- **Questions**: [GitHub Discussions](https://github.com/gawwr4v/RadialMenu/discussions)
- **Stack Overflow**: Tag your question with `radial-menu` and `android`

## Before opening an issue

1. Check the [README](README.md) for usage examples
2. Check [existing issues](https://github.com/gawwr4v/RadialMenu/issues)
3. Check the [CHANGELOG](CHANGELOG.md), your issue may be fixed in a newer version

## Response time

This is a community-maintained open source project.
Issues are typically reviewed within a few days.
PRs with tests and passing CI are prioritized.
